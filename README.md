# 結婚市場価値診断 — 婚活偏差値算出プロトタイプ

結婚相談所の無料入会相談への集客を目的とした、Web完結型の診断コンテンツです。

## デモ

12問・約3分の設問に回答すると、5つの軸で婚活力を可視化し、総合スコア（婚活偏差値）・ランク・強み/課題を表示します。

## 技術スタック

- **React 18** + **Vite 6**
- CSS-in-JS（インラインスタイル）
- SVGレーダーチャート（ライブラリ不使用）
- CSSアニメーション（桜の花びらエフェクト）

## セットアップ

```bash
npm install
npm run dev
```

`http://localhost:5173` でアクセスできます。

## ビルド

```bash
npm run build
```

`dist/` に静的ファイルが出力されます。

## 診断ロジック概要

### 5つの評価軸（各20点満点 / 合計100点）

| 軸 | 内容 |
|---|---|
| スペック力 | 年齢・年収・学歴等の客観条件 |
| 人間力 | コミュニケーション力・共感力 |
| 婚活リテラシー | 結婚観・条件の柔軟性・行動量 |
| ライフスタイル | 趣味・自己投資・生活習慣 |
| タイミング適性 | 婚活開始の適切さ・市場ポジション |

### ランク

| スコア | ランク | タイプ名 |
|---|---|---|
| 85〜100 | S | プラチナ婚活者 |
| 70〜84 | A | ゴールド婚活者 |
| 55〜69 | B | シルバー婚活者 |
| 40〜54 | C | ブロンズ婚活者 |
| 〜39 | D | スタート婚活者 |

### 男女分岐

Q1で性別を選択後、年齢・年収・学歴の配点ロジックが分岐します。

## Web表示 vs 来店提供の情報設計

| 要素 | Web | 来店 |
|---|---|---|
| 総合スコア・ランク | ◯ | ◯ |
| レーダーチャート | ◯ | ◯（詳細版） |
| 強み・課題の概要 | ◯ | ◯（詳細解説） |
| 相性の良い異性の人数 | 星表示のみ | **具体的人数** |
| 改善アクションプラン | ✕ | **◯** |
| 個別婚活プラン提案 | ✕ | **◯** |
