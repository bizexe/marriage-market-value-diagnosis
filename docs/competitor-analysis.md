# 結婚相談所 主要7社 集客コンテンツ・導線・SNS活用 比較分析

---

## 1. オーネット

### Web導線
- **入口コンテンツ**: 「結婚チャンステスト」（60秒の無料診断）がメインCTA。Web上で希望条件を入力するとマッチング人数が表示されるシミュレーション型
- **導線設計**: チャンステスト申込 → 結果通知（数日後に郵送/メール）→ 来店予約への誘導。「お急ぎの方へ」欄から直接来店予約も可能
- **来店体験**: 15分程度のサービス紹介動画視聴 → マッチングシミュレーション体験（実際の会員プロフィール表示）→ ヒアリング・プラン説明
- **テレビCM**: 大規模なテレビCMを継続投下し、認知獲得の太い柱としている。DayDay（日テレ）での取材放送実績あり

### SNS活用
- X（旧Twitter）、Instagramで婚活に関する啓発コンテンツを配信
- オウンドメディアでの婚活コラム記事によるSEO流入
- SNS活用は他社比で控えめ。マスメディア（テレビCM・交通広告）依存度が高い

### 強み
- 「チャンステスト」による心理的ハードルの低い初回接点設計。入会申込ではなくシミュレーションという建て付けで気軽さを訴求
- IBJ提携によるNo.1会員数の訴求力（約11.6万人）
- 全国に店舗網があり地方でも対応可能
- テレビCMによる圧倒的なブランド認知

### 弱み
- SNS・デジタルコンテンツでの情報発信が弱く、若年層へのリーチが限定的
- チャンステストから来店までの間にナーチャリングコンテンツが薄い（結果郵送の待ち時間に離脱リスク）
- 口コミサイトでの「勧誘がしつこい」という評判がCVRに影響する可能性

## 2. ツヴァイ

### Web導線
- **入口コンテンツ**: 「マッチング無料体験」と「婚活シミュレーション（今から始めていつ成婚するかわかる）」の2軸
- **導線設計**: 公式サイトTOP → 資料請求（パーソナル結婚診断）またはダイレクトに無料来店予約 → 店舗でのマッチングシミュレーション体験。資料請求→無料コンサルティングの2ステップ推奨
- **来店体験**: カラーテストによる価値観マッチングシミュレーション、マリッジコンサルタントによる1〜2時間の個別ヒアリング。満足度96%と訴求
- **オウンドメディア**: 「ウーマンツヴァイ」というコンテンツサイトを運営。宣伝色を排除した恋愛・結婚コラムで潜在層を獲得し、資料請求・無料コンサルティングの参照元トップに成長させた実績あり
- **割引施策**: Web予約割、20代割、のりかえ割、友達紹介（双方に1万円商品券）など多彩なキャンペーン

### SNS活用
- Instagram（成婚体験談、婚活Tips）での情報発信
- Facebook活用事例あり（Meta社のビジネス成功事例に選出歴）
- コンテンツマーケティング（SEO記事）が強く、検索エンジン流入が前年比7倍に成長した実績

### 強み
- オウンドメディア「ウーマンツヴァイ」による潜在層の囲い込みが秀逸。宣伝色を排除した婚活・恋愛コラムで潜在ニーズを喚起
- イオングループの信頼感とブランド力
- 全国53店舗の店舗網
- 割引施策が豊富で入会ハードルを下げる仕掛けが多い
- AIアナリスト導入等、データドリブンなWebマーケティング体制

### 弱み
- SNS（特にYouTube・TikTok）での動画コンテンツが弱く、ナレソメ予備校のような若年層へのバイラル集客ができていない
- ツヴァイ自体のブランド想起がオーネットほど強くない

## 3. IBJメンバーズ

### Web導線
- **入口コンテンツ**: 公式サイトTOPから「無料カウンセリング予約」への直接導線。資料請求も用意
- **導線設計**: 公式サイト → 無料カウンセリングWeb予約 → 2時間程度で電話確認 → 来店（個室ブースでの面談）。婚活パーティー参加後のアンケートから無料相談への誘導もあり
- **来店体験**: 完全個室での面談、成婚メソッドの紹介、会員データベースの説明。プロフィール例や成婚事例の提示
- **親会社IBJの知名度**: IBJ連盟全体としてのNo.1実績（会員数・成婚数）を前面に訴求

### SNS活用
- IBJ本体としての公式メディア運用。婚活パーティーPARTY☆PARTY、ブライダルネット等の関連サービスとの相互送客
- YouTube・SNSでの積極的な自社発信は限定的
- 比較サイト・口コミサイトへのPR記事出稿によるSEO対策が中心

### 強み
- 「IBJ」のブランド力。業界最大手プラットフォームの直営店という圧倒的な信頼感
- 会員数約8.5万人以上（IBJ連盟全体で10万人超）の規模
- 成婚メソッドの体系化。データに基づくアドバイスの品質
- 婚活パーティーからの導線（パーティー参加→アンケート→無料相談への自然なアップセル）

### 弱み
- 公式サイトの導線がやや分かりにくいとの声あり
- 店舗が主要都市11拠点のみで、地方に弱い
- SNS発信が弱く、デジタルネイティブ世代への訴求力が不足
- 料金が高めの印象があり、初回接点での価格訴求が弱い

## 4. サンマリエ

### Web導線
- **入口コンテンツ**: 公式サイトからの資料請求が主導線。無料で婚活テスト「本性診断レコメンド」を実施
- **導線設計**: 資料請求 → 資料郵送（3日程度で到着）→ 電話でのフォロー → 無料相談予約。オンライン無料相談にも対応
- **来店体験**: 専任カウンセラーによるヒアリング、婚活プラン個別設定、活動会員プロフィール提示
- **セミナー型コンテンツ**: 「マリカレ」（婚活カレッジ）としてファッション・メイク・会話等の各種婚活スキルアップセミナーを提供。これ自体が差別化コンテンツ

### SNS活用
- Instagram、Xで婚活コラム・成婚体験談を配信
- IBJグループ全体のメディア露出と連携
- 自社SNS運用はやや控えめ

### 強み
- 創業44年の老舗ブランド。長年の婚活ノウハウの蓄積
- 「マリカレ」による婚活スキルアップセミナーは他社にない差別化ポイント。「学びながら婚活」という付加価値
- 全国23店舗＋44サテライトで地方カバレッジが広い
- 手作業での特別紹介（担当者が人柄を理解した上でのマッチング）が訴求ポイント
- 出張カウンセリング対応

### 弱み
- 資料請求後の電話フォローが「しつこい」という口コミあり。CVRに悪影響
- SNS・デジタルマーケティングでの存在感が薄い
- IBJメンバーズとの差別化が分かりにくく、同グループ内でのカニバリゼーションの懸念
- Web導線がやや古典的（資料請求→電話→来店という従来型ファネル）

## 5. パートナーエージェント

### Web導線
- **入口コンテンツ**: 公式サイトからの「無料カウンセリング予約」と「資料請求」の2軸。成婚率No.1を前面に訴求
- **導線設計**: 公式サイト → 無料カウンセリング予約（来店/オンライン選択可）→ コンシェルジュによるカウンセリング。グラフ等の視覚的資料を用いた丁寧な説明が特徴
- **来店体験**: 婚活市場のデータ分析をグラフで見せながら具体的に説明。婚活力の客観的評価、厳しい指摘も含む率直なフィードバックが好評
- **独自調査コンテンツ**: 婚活に関するアンケート調査レポートを定期的に発信。PR記事としてメディア掲載獲得に活用
- **アプリ連携**: 「パートナーエージェントApp」をマッチングアプリ感覚の入口として提供（月額9,800円〜）。アプリからの本格入会へのアップセル導線

### SNS活用
- X: フォロワー数は控えめ。婚活調査データの発信
- Instagram: フォロワー約400人。婚活アドバイスやキャンペーン情報を投稿。LINE公式への誘導を実施
- YouTube: チャンネル保有。婚活ノウハウ動画を配信
- LINE公式アカウント: Instagramからの誘導先として活用。1to1コミュニケーション強化

### 強み
- 成婚率No.1という明確な数値実績の訴求力
- コンシェルジュの質の高さ（率直で具体的なフィードバック）が口コミで高評価
- コネクトシップという独自の会員相互紹介プラットフォーム
- 成婚退会後12ヶ月の無料フォローという独自サポート
- アプリ版による低価格帯からのアップセル導線

### 弱み
- SNSフォロワー数が非常に少なく（Instagram約400人）、デジタル上の存在感が極めて薄い
- IBJ連盟からの除名歴があり、業界内での立場がやや複雑
- コネクトシップの実質的な紹介可能会員数が公称値より少ない（約2.8万人）
- Web上のコンテンツマーケティングが弱い

## 6. ナレソメ予備校

### Web導線
- **入口コンテンツ**: YouTubeチャンネル（登録者36万人超）が集客の最大エンジン。婚活ノウハウ・成婚インタビュー・体験談を毎日更新
- **導線設計**: YouTube/TikTok/X視聴 → 公式サイト「無料個別相談」申込 → Zoomでのオンライン面談（約1時間）→ 戦闘力診断（婚活レベルの数値化）→ 入塾審査
- **無料相談の特徴**: 「婚活戦闘力診断」で自分の婚活市場価値を33点満点で数値化。入会前に現実的なフィードバックを提供。他社では得られない「厳しくも正直な評価」が差別化ポイント
- **入会審査**: 選考制を採用し「選ばれた人だけが入れる」ブランディング。これ自体が希少性の演出になっている

### SNS活用（業界最強のSNSマーケティング）
- **YouTube**: 登録者36万人超。毎日更新。成婚インタビュー、婚活ノウハウ、代表・勝倉氏のマジレスコンテンツが人気。婚活リアリティ風の演出で視聴者を惹きつける
- **TikTok**: ショート動画で若年層にリーチ。YouTubeコンテンツの切り抜きも活用
- **X（旧Twitter）**: フォロワー約8,800人。代表の勝倉氏の個人アカウントが実質的なメインメディア。歯に衣着せぬ発言が時に炎上するも、認知拡大には大きく貢献
- **会員コミュニティ**: 会員同士がXで繋がるコミュニティが自然発生。UGC（ユーザー生成コンテンツ）として口コミ拡散に寄与
- **オフラインイベント**: 女子会（80名規模）等のリアルイベントをSNSで拡散

### 強み
- **SNSマーケティングが圧倒的**: YouTube・TikTok・Xを組み合わせた集客力は業界随一。広告費をかけずにオーガニック流入を大量獲得
- 「予備校」というコンセプトの明確さ。学ぶ→実践→成婚という分かりやすいストーリー
- 戦闘力診断という独自フックで無料相談のCVRが高い
- 入会審査による選考制が「ここに入れる＝自分のレベルが認められた」という心理的満足を生む
- 20〜30代の若年層に圧倒的に強い（平均年齢31歳）
- 成婚後サポート「イトナミ」による卒業生コミュニティが口コミ再生産装置として機能

### 弱み
- 代表の過激な発信が一部で反感を買い、ネガティブ口コミも多い
- 入会審査で断られるケースがあり、間口が狭い
- 入会から活動開始まで3〜4ヶ月の待機期間が発生するケースあり
- 40代以上には合わない（若年層特化のため）
- SNSでの競合他社否定コンテンツが業界内で問題視される場合あり

## 7. エン婚活エージェント

### Web導線
- **入口コンテンツ**: 「無料マッチングシミュレーション」がメインフック。来店不要で全てオンライン完結
- **4つの無料体験ツール**:
  1. 求婚者チェッカー（自分と結婚したい人の人数が分かる）
  2. お試し検索（希望条件に合う会員の人数＋プロフィール一部閲覧）
  3. パーソナル活動プランニング（個別の婚活計画策定）
  4. 結婚価値観診断（EQベースの102問の本格診断）
- **導線設計**: 公式サイト → 無料シミュレーション会員登録（メールのみ）→ 各種ツール体験 → 有料会員申込。電話・来店勧誘なしを明示
- **全額返金保証**: 3ヶ月以内にコンタクト不成立なら全額返金。リスクリバーサルでCVR向上

### SNS活用
- エン・ジャパングループの企業ブランドを活用した信頼性訴求
- 自社SNSでの積極的な発信はやや控えめ
- 比較サイト・口コミサイトでのSEO施策が中心
- 約50種類のオンライン婚活講座を会員向けに提供（コンテンツマーケティング資産）

### 強み
- **圧倒的な入口の低さ**: 来店不要・電話勧誘なし・オンライン完結。無料体験だけで帰れる設計が他社と根本的に異なる
- 無料体験ツール群の充実度が高く、入会前の意思決定に必要な情報を自分で確認できる
- 年間18万円前後という業界最安水準の料金。成婚料も無料
- 全額返金保証制度によるリスクゼロ訴求
- エン・ジャパン（東証プライム上場）の運営という信頼感
- 初月会費無料等のキャンペーンを定期的に実施

### 弱み
- オンライン完結型のため対面サポートが受けられない
- コネクトシップ経由の会員が中心で、IBJ連盟の会員にはアクセスできない
- 地方によっては紹介可能会員数が少ない
- SNS上での存在感・話題性が薄く、認知獲得チャネルがSEO・比較サイト依存
- 自主的に動ける人向けで、手取り足取りのサポートを求める層には不向き

## 自社への示唆（集客コンテンツ設計のポイント）

### 1. 無料体験の「価値提供ファースト」設計が最重要
エン婚活の4種ツールとナレソメの戦闘力診断が示す通り、「入会判断に必要な情報を事前に無料で得られる」設計が現代の集客の鍵。単なるサービス説明ではなく、ユーザー自身の市場価値や可能性を「可視化」するツールが効果的。

### 2. SNS（特にYouTube・TikTok）は未開拓ブルーオーシャン
ナレソメ予備校が唯一本格的にSNSを活用し圧倒的成果を出している。大手6社はいずれもSNS活用が弱く、ここに参入余地が大きい。婚活ノウハウ・成婚体験談・Q&A等の動画コンテンツは、広告費をかけずに潜在層にリーチできる最も費用対効果の高いチャネル。

### 3. 「心理的ハードルの段階的除去」が導線設計の核
成功している導線は「診断・シミュレーション」→「具体的なデータ提示」→「個別相談」と段階を踏む。いきなり来店予約ではなく、自分で試せる無料ツールを入口にすることでCVRが大幅に向上する。

### 4. ネガティブ口コミ対策が集客の前提条件
サンマリエの「電話勧誘がしつこい」、オーネットの「勧誘が怖い」等の口コミは集客の最大の阻害要因。エン婚活の「電話・来店勧誘なし」明示やナレソメの「無理な引き留めなし」の姿勢が好評価を得ている点は重要な示唆。
